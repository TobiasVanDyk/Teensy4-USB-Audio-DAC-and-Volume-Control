# Teensy4-USB-Audio-DAC-and-Volume-Control
Teensy4 USB Audio DAC and Volume Control using Volume Up and Down buttons (Teensy4USBAudioKeybrd2.ino) and with an additional rotary encoder as volume control (Teensy4USBAudioKeybrd6.ino) for the details). The encoder push-button switch the display off (the mute function does function for the Teensy 4 USB Audio object). It is also convenient to attach a pushbutton to the Teensy 4 on/off pin - pressing it for 5 seconds will switch the DAC off and pressing it again for 2 seconds will switch it back on.
